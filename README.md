# 👟 SportSteps - Tienda de Calzado Deportivo

**SportSteps** es una plataforma  especializada en calzado de alto rendimiento. 

---

## 🚀 Características principales

* **Diseño:** Interfaz oscura con paleta de colores Verde Neón (`#AAFF00`) y Cian (`#3fdae6`).
* **Filtrado Dinámico:** Navegación por categorías (Running, Casual, Fútbol, Básquet) y género.
* **Carrito de Compras Lateral:** Panel interactivo para gestionar productos en tiempo real.
* **Diseño Responsivo:** Optimizado para dispositivos móviles, tablets y escritorio mediante Bootstrap 5.
---

## 🛠️ Tecnologías utilizadas

* **HTML5** - Estructura semántica.
* **CSS3** - Estilos personalizados con variables y efectos.
* **JavaScript** - Lógica de base de datos simulada, filtrado y carrito de compras.
* **Bootstrap 5** - Sistema de grillas, componentes y componentes interactivos.
* **Google Fonts** - Tipografías *Bebas Neue* y *Montserrat*.
* **Bootstrap Icons** - Iconografía deportiva y de interfaz.
